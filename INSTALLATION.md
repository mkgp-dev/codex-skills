# Skill Installation

Copy and paste the commands below with the Vercel `skills` CLI.

## List Available Skills

```bash
npx skills add mkgp-dev/codex-skills --list
```

## Install A Skill

### `zod-best-practices`

```bash
npx skills add mkgp-dev/codex-skills --skill zod-best-practices
```

### `react-hook-form-with-zod`

```bash
npx skills add mkgp-dev/codex-skills --skill react-hook-form-with-zod
```

### `zustand-state-management`

```bash
npx skills add mkgp-dev/codex-skills --skill zustand-state-management
```

## Install For Codex

### `zod-best-practices`

```bash
npx skills add mkgp-dev/codex-skills --skill zod-best-practices -a codex
```

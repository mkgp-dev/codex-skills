# Skill Installation

Copy and paste the commands below with the Vercel `skills` CLI.

## List Available Skills

```bash
npx skills add mkgp-dev/codex-skills --list
```

## Install A Skill

### `zod-best-practices`

```bash
npx skills add mkgp-dev/codex-skills --skill zod-best-practices
```
